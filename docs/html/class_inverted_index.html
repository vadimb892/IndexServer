Детальний опис

Представляє інвертований індекс і містить для цього усі необхідні компоненти.

Отримує пари запит - список_ресурсів і записує їх у відповідних таблицях. Видає за окремими ключами список джерел із відповідних таблиць, виконує пошук також у записаних на носії секціях, містить набір функцій для обробки виключень.
