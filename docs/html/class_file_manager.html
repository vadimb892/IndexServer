Детальний опис

Керує вмістом вказаної папки за шляхом FileManager::storingPath, для цього: 1, збирає інформацію про всі файли у папці за допомогою FileManager::readSectionsInfo(), FileManager::createSectionsInfo(), 2, підраховує за типом файли із допомогою методу FileManager::countFilesOfType().

3, зменшує об'єм, що займають дані шляхом пріоритетного очищення у методі FileManager::decreaseStoredData(), FileManager::clearMemory().
