Інструкція перегляду документації:
1. У папці html є файл index.html. Потрібно зайти до нього із браузеру.
2. Відкриється титульна сторінка. Обираємо категорію "Класи" і знову підпункт "Класи".
3. Далі можна переглядати вміст кожного класу.

Документація згенерована в Doxygen для с++

Інструкція збірки документації:
1. Встановлюємо Doxygen.
2. Розмічуємо код.
3. Ствроюємо папку, де буде документація.
4. Переходимо до нової папки у командній строці.
5. Виконуємо "doxygen -g <назва файлу конфігурації>".
6. Утворюється файл конфігурації. Налаштовуємо його зміст.
7. Виконуємо "doxygen <назва  файлу конфігурації>.
8. Готово.
